# frontend-design

打造**独特、可上线级别**的前端界面,用扎实的代码实现细腻的设计;主动避开"AI 千篇一律"的观感。

## 动手前先想清楚四件事
- **Purpose 目的**:解决什么问题、给谁用。
- **Tone 调性**:一个大胆明确的美学方向(brutalist / maximalist / retro-futuristic / luxury / 手作质感 …)。
- **Constraints 约束**:技术限制。
- **Differentiation 记忆点**:"别人会记住的那一个东西是什么?"
> 重在 intentionality(有意图)而非 intensity(堆强度)。大胆的极繁和克制的极简都行,关键是统一与刻意。

## 五个发力点
1. **Typography**:选漂亮、独特、有性格的字体。**别用 Arial / Inter / Roboto** 这类通用字。
2. **Color & Theme**:统一方向 + 主色 + 锐利的强调色;2–3 色为宜,别彩虹。
3. **Motion**:CSS 动效,要有高光时刻和有质感的微交互。
4. **Spatial Composition**:非对称、重叠、对角线流向、打破网格。
5. **Visual Details**:用渐变/纹理/图案/自定义效果营造氛围。

## 明确避免
通用 AI 观感:烂大街字体、俗套配色(尤其白底/深底上的紫渐变)、可预测的卡片布局、缺乏语境性格的模板感设计。
